Painel de Configuração Fedora

Gerencie e automatize as instalações do seu sistema de forma visual

👋 Bem-vindo ao Painel de Automação!

Antes de prosseguir com as etapas de configuração e instalação de aplicativos, é altamente recomendado realizar uma atualização completa no seu sistema operacional. Isso garante a estabilidade dos pacotes e evita conflitos.

Atualização Geral do Fedora

Atualiza o índice de repositórios, sincroniza as chaves de segurança e baixa as versões mais recentes de todos os componentes do sistema.

⚙️ Etapa 1: Otimização do Sistema e Idioma

Melhorar velocidade de download do DNF

Configura os downloads paralelos simultâneos e ativa a busca automática pelo espelho de servidor mais rápido no gerenciador de pacotes.

Configuração de Idioma e Localidade (PT-BR)

Instale os pacotes de tradução, o corretor ortográfico e altere a localidade padrão do sistema operacional de forma modular usando os botões abaixo:

⚠️ Aviso: Reinicie o computador após concluir o passo 3 para aplicar as alterações de idioma de forma global!

Corrigir problemas de data/hora em Dual-Boot (Windows/Linux)

Configura o relógio do hardware para usar o padrão UTC, evitando que o Windows e o Linux desorganizem os horários mutuamente ao alternar entre os sistemas.

📦 Etapa 2: Repositórios, Codecs e Aceleração Gráfica

Ativar Repositórios RPM Fusion (Livre e Não-Livre)

Instala e sincroniza as fontes de pacotes comunitários essenciais, adiciona os dados para a loja do KDE e aplica uma correção de segurança automática caso o repositório Rawhide tente se ativar indevidamente.

Habilitar Suporte Completo ao Flatpak (Flathub) no KDE

Adiciona o repositório global Flathub, garante a instalação da engine do Flatpak, integra o suporte visual diretamente à loja Discover do KDE e instala a ferramenta Flatseal para gerenciar permissões de aplicativos.

Instalar Suporte a Arquivos Compactados (7z, RAR e ZIP)

Instala os utilitários de linha de comando e bibliotecas necessárias para extrair e compactar arquivos nos formatos zip, rar, 7z e tar pelo gerenciador de arquivos.

Instalar Codecs Multimídia Essenciais

Substitui a versão limitada do ffmpeg pela versão completa sem restrições de patentes, instala pacotes de codecs gstreamer sem dependências fracas inúteis e ativa o Cisco OpenH264.

Atualizar Firmwares de Hardware

Força a atualização dos índices do gerenciador de firmwares do sistema, mapeia os dispositivos e periféricos locais e aplica atualizações de estabilidade e microcódigos da CPU ou firmwares Linux gerais.

Extras e DVD Play (Repositórios Tainted)

Habilita os sub-repositórios restritos para habilitar criptografia de reprodução de DVDs de vídeo comerciais e injetar pacotes extras de firmwares proprietários de hardware travados.

VA-API e Codecs para Aceleração Gráfica

Instala drivers e utilitários de aceleração de vídeo em hardware para placas de vídeo AMD e Intel no Fedora. Use o botão principal. Use o botão de contingência apenas se o primeiro acusar conflito de pacotes.

🔤 Etapa 3: Fontes para Compatibilidade

Instalar fontes Microsoft para compatibilidade

Instala o pacote completo de fontes TrueType da Microsoft (como Arial, Times New Roman, Calibri, entre outras). Essencial para garantir que documentos de texto, planilhas e páginas da web abram com a formatação correta.

🎮 Etapa 4: Configuração para Jogos (Gaming)

Drivers Vulkan Completos (Para GPUs AMD)

Instala as ferramentas e bibliotecas Vulkan em versões de 32 e 64 bits. Indispensável para o Proton converter jogos de Windows para o Linux com alto desempenho.

Instalar Steam e Regras de Dispositivos

Instala a plataforma oficial da Steam no sistema junto com os pacotes de regras de udev necessários para o reconhecimento de manetes e óculos de VR.

🎬 Etapa 5: Produção de Áudio, Vídeo e Apps

Instalação de Aplicativos Recomendados (Via Central de Software / Discover)

Para uma melhor experiência no Fedora KDE, clique no botão abaixo para abrir a Central de Software e procure manualmente pelos seguintes softwares recomendados:

  • Haruna (Player de vídeo moderno feito para o KDE)
  • OnlyOffice (Suíte de escritório com máxima compatibilidade com o MS Office)
  • Opera (Navegador focado em produtividade e recursos extras)
  • OBS Studio (A principal ferramenta para gravação e transmissão de tela)
  • Krita (Pintura digital e edição de imagens rasterizadas)
  • KDEnLive / ShotCut (Editores de vídeo profissionais e estáveis)

Ativar Câmera Virtual no OBS Studio

Instala as ferramentas e o módulo v4l2loopback do kernel para permitir que o OBS crie uma webcam virtual reconhecida por apps de conferência (como Meet, Teams e Discord).

⚠️ Importante: Reinicie o OBS após instalar. Este recurso exige que o Secure Boot da sua placa-mãe esteja configurado com chaves corretas ou desativado para carregar o módulo de kernel.

Melhoria de Áudio com EasyEffects

Instala o melhor processador e equalizador de efeitos de som para servidores PipeWire, ideal para aplicar redução de ruído em microfones e melhorias de graves nas caixas de som.

💡 Nota: Nunca altere manualmente o dispositivo padrão de som do sistema para "Easy Effects Sink" nas configurações do KDE. Deixe o aplicativo trabalhar de forma invisível em segundo plano.
🛠️ Etapa 6: Manutenção e Tarefas Avançadas

Limpar Arquivos Temporários e Resíduos

Remove os arquivos de cache baixados pelo DNF, desinstala dependências órfãs que nenhum programa mais utiliza e remove runtimes antigos e não usados do Flatpak.

Gerenciamento de Kernels Antigos (Opcional)

Lista todos os kernels atualmente instalados na sua máquina para que você possa remover manualmente versões antigas e liberar espaço na partição /boot.

💡 Instrução: O botão listará os kernels. Se desejar apagar algum antigo, digite no terminal secundário: sudo dnf remove nome-do-kernel

Upgrade de Versão do Fedora

Prepara o sistema para migrar para a próxima grande versão do Fedora de forma totalmente segura. Nenhuma das modificações feitas por este guia impede este processo automático.

Migrar do Canal Beta para o Estável (Release)

Sincroniza os pacotes locais com o repositório estável de produção do Fedora, fazendo o downgrade ou upgrade necessário dos componentes modificados na versão Beta.

⚠️ Aviso: É obrigatório reiniciar o computador imediatamente após concluir a sincronização dos pacotes.
